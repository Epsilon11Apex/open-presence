🚀 Open Presence (PreMiD Clone)
Une alternative ultra-légère à PreMiD, développée en Rust (Tauri) et TypeScript pour optimiser les performances et la consommation RAM.

✨ Fonctionnalités
Performance : Utilise le moteur Rust pour un impact système minimal.

Confidentialité : 100% local, aucune donnée n'est envoyée sur des serveurs tiers.

YouTube Support : Synchronisation automatique du titre de la vidéo sur ton profil Discord.

🛠️ Installation rapide
1. Le Bridge (Logiciel)
Télécharge le dernier .exe ou .msi dans les Releases.

Lance l'application. Elle doit rester ouverte pour mettre à jour ton statut.

2. L'Extension (Navigateur)
Télécharge le dossier extension-presence de ce dépôt.

Accède à chrome://extensions/ dans ton navigateur.

Active le Mode développeur.

Clique sur Charger l'extension non empaquetée et sélectionne le dossier.

🏗️ Développement (Build)
Si tu souhaites compiler le projet toi-même :

PowerShell
# Installer les dépendances
npm install

# Lancer en mode dev
npm run tauri dev

# Créer l'exécutable optimisé
npm run tauri build
